Density Analysis

The following series of charts display anticipated employment and dwelling unit densities by year in Polk County, as contained in the Polk TPO 2045 LRTP Land Use Forecast. These plots reveal how the exiting and future year densities compare to minimum BRT thresholds in “A Framework for Transit Oriented Development in Florida.” In addition, mean corridor typology values from “TCRP 187: Livable Transit Corridors: Methods, Metrics, Strategies”, and “DRPT Multimodal Design Guidelines” are included for the employment charts. Similar values are not avaialbe for dwelling unit figures, as TCRP 187 uses population as a metric.

Methadology note: jobs and dwelling units were estimated using a area-weighted interpolation process, which assumes jobs and housing are equally spread across each TAZ. Densities were calculated using land area for each TAZ, which was calculated by removing all area in each TAZ designated as water by FDEP. This method can be enhanced to account for buildign size, type, or other information that can improve the disaggregation.
