Lakeland TOD Analysis

This website reviews the findings of a TOD analysis that explores base and future land use conditions at station, corridor, and regional perspectives. These conditions are compared to thresholds identified in “A Framework for Transit Oriented Development in Florida.”. Further, the analysis explores how much growth anticipated in the county the corridor would need to absorb in order to reach TOD thresholds for BRT. This scenario is then compared to the model forecast.
